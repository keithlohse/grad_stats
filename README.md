# grad_stats
 Graduate statistics course covering basic estimation into more complex crossed and nested models. All analyses are taught using a model comparison approach.
 
 Please note that all of the materials in this repository are supporting materials for a graduate statistics class taught by Keith Lohse at the University of Utah.
 
 Video lectures and recent copy of the syllabus are available from: https://sites.google.com/site/lohsekr/design-and-analysis-i
